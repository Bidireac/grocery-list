## General info
Using the React forms, you can add groceries and their respective quantity, modify them and also delete them.
	
## Technologies
Project is created with:
* HTML 5
* CSS 3
* Javascript ES6
* React ^17

## Setup
To run this project, open the file location in the terminal and run npm start.
